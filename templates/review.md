# 코드 리뷰 템플릿

## 사용법

Claude Code에서 아래 프롬프트의 `{{변수}}`를 실제 값으로 바꿔서 입력한다.

## 프롬프트

```
다음 코드를 리뷰하기 위한 에이전트 팀을 만들어줘.

**리뷰 대상**: {{PR 번호 또는 파일/디렉토리 경로}}
**리뷰 목적**: {{일반 리뷰, 보안 감사, 성능 최적화 등}}
**핵심 관심사**: {{특별히 주의할 부분}}

팀 구성:
1. reviewer 에이전트 타입으로 "보안리뷰어" 팀원을 스폰해줘 — 보안 취약점을 집중 검토한다.
2. reviewer 에이전트 타입으로 "성능리뷰어" 팀원을 스폰해줘 — 성능 이슈를 집중 검토한다.
3. reviewer 에이전트 타입으로 "품질리뷰어" 팀원을 스폰해줘 — 코드 품질, 테스트 커버리지를 검토한다.

세 명이 동시에 각자의 관점으로 리뷰하고, 결과를 종합해줘.
```

## 태스크 흐름

```
[보안리뷰어: 보안 검토]   ──┐
[성능리뷰어: 성능 검토]   ──┼──→ [리드: 결과 종합 및 우선순위 정리]
[품질리뷰어: 품질 검토]   ──┘
```
